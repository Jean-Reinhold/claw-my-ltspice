# Practical Op-Amp Differentiator

![Rendered practical differentiator schematic](preview.svg)

This example demonstrates a practical inverting differentiator using the
repo-owned `CLAW_IDEAL_OPAMP` macromodel. The input capacitor provides the
differentiating action, the small series resistor limits input-side high-frequency
gain, and the feedback capacitor limits the feedback bandwidth.

Expected behavior:

- A finite-slope `200 Hz` pulse train produces bounded positive and negative
  output pulses at the input transitions.
- The feedback resistor keeps the inverting node biased at DC, and the feedback
  capacitor prevents the ideal op amp from amplifying arbitrarily fast edges.
- The committed source netlist includes `../../lib/claw_opamps.lib`; generated
  builds copy the op-amp library beside the output files and include
  `claw_opamps.lib`.

Commands:

```bash
./claw-spice code build examples/transient/opamp-practical-differentiator/opamp_practical_differentiator.py
./claw-spice sim run examples/transient/opamp-practical-differentiator/opamp_practical_differentiator.cir
./claw-spice raw plot runs/latest/opamp_practical_differentiator.raw V(in) V(out) --output runs/latest/opamp_practical_differentiator.svg
./claw-spice show examples/transient/opamp-practical-differentiator/opamp_practical_differentiator.asc
```

The `.meas` directives report `vout_max`, `vout_min`, and `vout_rms` over the
settled transition window.

`preview.svg` is generated by the real `ltspice_to_svg` renderer when you run
`./claw-spice examples render` or `./claw-spice docs assets`.
