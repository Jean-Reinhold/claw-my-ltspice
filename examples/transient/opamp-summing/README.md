# Generic Op-Amp Inverting Summing Amplifier

Transient feedback example that sums two stepped inputs with different resistor
weights using the repo-owned `CLAW_IDEAL_OPAMP` macromodel.

```bash
./claw-spice code build examples/transient/opamp-summing/opamp_summing.py
./claw-spice sim run examples/transient/opamp-summing/opamp_summing.cir
./claw-spice raw plot runs/latest/opamp_summing.raw V(in1) V(in2) V(out) --output runs/latest/opamp_summing.svg
./claw-spice show examples/transient/opamp-summing/opamp_summing.asc
```

Expected behavior: `V(out)` approximately follows `-(2 * V(in1) + V(in2))`
within the available output swing.

`preview.svg` is generated by the real `ltspice_to_svg` renderer when you run
`./claw-spice examples render` in the Docker/runtime environment.
