# Generic Op-Amp Difference Amplifier

Transient difference-amplifier example using matched resistor ratios and the
repo-owned `CLAW_IDEAL_OPAMP` macromodel.

```bash
./claw-spice code build examples/transient/opamp-difference/opamp_difference.py
./claw-spice sim run examples/transient/opamp-difference/opamp_difference.cir
./claw-spice raw plot runs/latest/opamp_difference.raw V(plus) V(minus) V(out) --output runs/latest/opamp_difference.svg
./claw-spice show examples/transient/opamp-difference/opamp_difference.asc
```

Expected behavior: `V(out)` approximately follows `2 * (V(plus) - V(minus))`.

`preview.svg` is generated by the real `ltspice_to_svg` renderer when you run
`./claw-spice examples render` in the Docker/runtime environment.
