# Generic Op-Amp Inverting Amplifier

Transient feedback example with gain approximately `-Rf/Rin = -2` using the
repo-owned `CLAW_IDEAL_OPAMP` macromodel.

```bash
./claw-spice code build examples/transient/opamp-inverting/opamp_inverting.py
./claw-spice sim run examples/transient/opamp-inverting/opamp_inverting.cir
./claw-spice show examples/transient/opamp-inverting/opamp_inverting.asc
```

Expected behavior: a 1 V input step produces approximately -2 V at the output
after settling.

`preview.svg` is generated by the real `ltspice_to_svg` renderer when you run
`./claw-spice examples render` in the Docker/runtime environment.
