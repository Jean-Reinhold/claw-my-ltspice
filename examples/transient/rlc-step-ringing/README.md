# Passive RLC Step Ringing

![Rendered RLC step schematic](preview.svg)

This example shows an underdamped passive second-order response. A `5 V` step
drives a `47 Ohm`, `10 mH`, `100 nF` series/shunt RLC network with a light
`10 kOhm` load. The ideal resonance is near `5 kHz`, so the transient should
ring for several cycles before settling near the final DC value.

Expected behavior:

- `V(out)` overshoots the `5 V` final value and rings at approximately the LC
  natural frequency.
- The `47 Ohm` series resistor limits current and keeps the transient numerics
  well behaved.
- The layout uses explicit wiring from the source through the resistor and
  inductor into the shunt capacitor/load output node.

Commands:

```bash
./claw-spice code build examples/transient/rlc-step-ringing/rlc_step_ringing.py
./claw-spice sim run examples/transient/rlc-step-ringing/rlc_step_ringing.cir
./claw-spice raw plot runs/latest/rlc_step_ringing.raw V(in) V(out) --output runs/latest/rlc_step_ringing.svg
./claw-spice show examples/transient/rlc-step-ringing/rlc_step_ringing.asc
```

Measurements:

- `ring_peak`: highest output voltage during the initial ringing window.
- `ring_min`: lowest output voltage during the initial ringing window.
- `vout_3ms`: late transient output voltage after the ringing has mostly
  decayed.

`preview.svg` is generated by the real `ltspice_to_svg` renderer when you run
`./claw-spice examples render` or `./claw-spice docs assets`.
