# Generic Op-Amp Voltage Follower

Transient op-amp feedback example using the repo-owned `CLAW_IDEAL_OPAMP`
macromodel. This validates a unity-gain buffer without depending on a vendor
op-amp model license.

```bash
./claw-spice code build examples/transient/opamp-voltage-follower/opamp_voltage_follower.py
./claw-spice sim run examples/transient/opamp-voltage-follower/opamp_voltage_follower.cir
./claw-spice show examples/transient/opamp-voltage-follower/opamp_voltage_follower.asc
```

Expected behavior: `V(out)` follows `V(in)` with small tracking error after the
initial step settles.

`preview.svg` is generated by the real `ltspice_to_svg` renderer when you run
`./claw-spice examples render` in the Docker/runtime environment.
