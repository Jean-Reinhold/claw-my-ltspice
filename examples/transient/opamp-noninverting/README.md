# Generic Op-Amp Non-Inverting Amplifier

Transient feedback example with gain approximately `1 + Rf/Rg = 3` using the
repo-owned `CLAW_IDEAL_OPAMP` macromodel.

```bash
./claw-spice code build examples/transient/opamp-noninverting/opamp_noninverting.py
./claw-spice sim run examples/transient/opamp-noninverting/opamp_noninverting.cir
./claw-spice show examples/transient/opamp-noninverting/opamp_noninverting.asc
```

Expected behavior: a 1 V input step produces approximately 3 V at the output
after settling.

`preview.svg` is generated by the real `ltspice_to_svg` renderer when you run
`./claw-spice examples render` in the Docker/runtime environment.
