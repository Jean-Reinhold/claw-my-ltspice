# Precision Half-Wave Rectifier

![Rendered precision rectifier schematic](preview.svg)

This example demonstrates an active precision rectifier built from the
repo-owned `CLAW_IDEAL_OPAMP` macromodel, an inline repo-owned diode model, and a
load resistor. The op-amp compensates the diode drop while the diode disconnects
the output node during negative input half-cycles.

Expected behavior:

- `V(rect)` follows the positive half-cycles of `V(in)` with much less diode-drop
  error than a passive rectifier.
- `V(rect)` returns close to ground during negative half-cycles through `RLOAD`.
- `vrect_max`, `vrect_min`, and `vrect_avg` summarize the steady rectified
  waveform over the measured window.

Commands:

```bash
./claw-spice code build examples/transient/precision-rectifier/precision_rectifier.py
./claw-spice sim run examples/transient/precision-rectifier/precision_rectifier.cir
./claw-spice raw plot runs/latest/precision_rectifier.raw V(in) V(rect) --output runs/latest/precision_rectifier.svg
./claw-spice show examples/transient/precision-rectifier/precision_rectifier.asc
```

The schematic preview is generated by `ltspice_to_svg`; missing symbols or
flag-only output should be treated as a rendering failure, not accepted as a
valid preview.
