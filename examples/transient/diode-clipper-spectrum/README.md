# Diode Clipper Harmonic Spectrum

![Rendered diode clipper schematic](preview.svg)

This example demonstrates the spectrum impact of a simple nonlinear passive
waveshaper. A `1 kHz`, `3 V` sine wave drives a shunt silicon diode clipper
through a `1 kOhm` source resistor. The diode uses an inline repo-owned
`DCLAW` model rather than a vendor model.

Expected behavior:

- Positive peaks are compressed near the diode forward voltage while negative
  peaks remain mostly set by the source and load divider.
- The asymmetric clipping introduces harmonic content visible with an FFT of
  `V(clip)`.
- The `.asc` is explicitly routed with `WIRE`, `FLAG`, and `IOPIN` records so
  the rendered schematic should look like a small shunt clipper circuit.

Commands:

```bash
./claw-spice code build examples/transient/diode-clipper-spectrum/diode_clipper_spectrum.py
./claw-spice sim run examples/transient/diode-clipper-spectrum/diode_clipper_spectrum.cir
./claw-spice raw plot runs/latest/diode_clipper_spectrum.raw V(in) V(clip) --output runs/latest/diode_clipper_spectrum.svg
./claw-spice raw fft runs/latest/diode_clipper_spectrum.raw V(clip) --output runs/latest/diode_clipper_spectrum_fft.svg
./claw-spice show examples/transient/diode-clipper-spectrum/diode_clipper_spectrum.asc
```

Measurements:

- `clip_max`: maximum clipped voltage after startup.
- `clip_min`: minimum clipped voltage after startup.
- `clip_avg`: average clipped voltage after startup, useful for showing the DC
  shift from asymmetric clipping.

`preview.svg` is generated by the real `ltspice_to_svg` renderer when you run
`./claw-spice examples render` or `./claw-spice docs assets`.
